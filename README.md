# рҹҡҖ Getting Started

## Clone Repository

```bash
git clone https://github.com/Noor1246/SonicSync.git
```

Move into project

```bash
cd SonicSync
```

Install frontend

```bash
npm install
```

Install backend

```bash
cd server
npm install
```

Create `.env` inside the **server** folder

```env
PORT=8000
MONGO_URI=YOUR_MONGODB_URI
JWT_SECRET=YOUR_SECRET
```

Run backend

```bash
cd server
npm start
```

Run frontend

```bash
npm run dev
```
